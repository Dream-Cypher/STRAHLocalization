# Summer Time Rendering: Another Horizon — English Localization

Documentation for the **English fan-localization patch** of the Nintendo
Switch visual novel "Summer Time Rendering: Another Horizon".

For the patch itself, installation instructions, and credits, see the
[**project README**](https://github.com/Dream-Cypher/STRAHLocalization#readme)
on GitHub.

## Pages

- [**TIPS Reference**](TIPS_reference.html) — the complete, unabridged text of
  every TIPS encyclopedia entry. A handful of the longest entries are
  shortened in-game (marked `[Summarized - see notes]`); this page always
  shows the full original wording.

  > ⚠️ **Contains spoilers** — only read as a reference for summarized
  > in-game entries as needed.

- [**Build Guide**](build.html) — step-by-step instructions for building the
  patch from source.

- [**Glossary**](glossary.html) — official name readings and terminology used
  throughout the translation.
